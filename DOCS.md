# ONC SPS — Shepherding Placement System

Internal tool for the Overcomers Nation Church Shepherding Placement System (SPS).

## What This Is

A lightweight web app that lets SPS shepherds log their weekly follow-up 
reports from their phone — no app download needed, just a link.

Reports are submitted to a Google Sheet via Apps Script and monitored 
live from the admin dashboard.

## Pages

- `/` — Shepherd reporting form (mobile)
- `/admin.html` — Admin dashboard (PIN protected)

## Stack

- HTML/CSS/JS — frontend
- Google Apps Script — backend API
- Google Sheets — data storage
- GitHub Pages — hosting

## Status

Phase 0 — Live and in use.
Phase 1 (React + Firebase) in development.

## Notes

Built for internal church use. Not open for external contributions.
